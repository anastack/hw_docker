# Домашнее задание по Docker (Генератор и Отчёт)

Тематика моего проекта — студенты. Скрипт генерирует список характеристик студентов(возраст, gpa, посещаемость, оп), потом делает из него HTML-отчёт. 


## Как запустить проект и получить отчёт

Если в GitHub Codespaces, откройте терминал в папке проекта и по очереди введите эти команды:

собираем образы:
`./run.sh build_generator`
`./run.sh build_reporter`

запускаем генератор:
`./run.sh run_generator`

запускаем отчет:
`./run.sh run_reporter`

## Как посмотреть отчёт в браузере

Чтобы открыть отчёт в браузере, нам нужно поднять веб-сервер (я использую nginx).
Вот как работает моя цепочка:
1. **Слой контейнера:** Команда запускает веб-сервер *nginx*, внутри которого на 80-м порту раздаётся файл `report.html`.
2. **Слой хоста Codespaces:** В команде запуска я использую флаг `-p 8080:80`. Это пробрасывает 80-й порт из контейнера на 8080-й порт виртуальной машины Codespaces. 
3. **Слой локального браузера:** Чтобы прокинуть порт из Codespaces на компьютер мы используем Ports в самом vs code.

Выполните команду:
`./run.sh report_server`

**Как открыть ссылку в Codespaces:**
1. В нижней панели VS Code откройте вкладку "Ports"
2. Найдите в списке порт 8080
3. Нажмите на "Open in Browser"
4. Откроется новая вкладка в браузере, и вы увидите отчет

---
*Дополнительные команды:*
Если захотите всё почистить (удалить сгенерированные файлы), введите: `./run.sh clear_data`
